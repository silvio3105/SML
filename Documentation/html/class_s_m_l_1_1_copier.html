Detailed Description

Copier class.

Wrapper for blocking or non-blocking copy process. User can easly switch from blocking copy process(eg., with standard memcpy function) to non-blocking process(eg., DMA copy process).

In case of non-blocking process, user must release process semaphore with Copier::done method.
