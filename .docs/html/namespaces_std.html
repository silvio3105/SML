Scan C-string and finds start of wanted parameter.

This function is replacment for scanf function. C-string format must be known to use this function. Function does not modify C-string. For example C-string: test1,test2,hello,test123 and call sScan(string,',',2,output), output.data will be pointing at letter 'h' and output.len will be 5. Parameter 2 because there are two separators before wanted token. output.len is 5 because text hello is 5 letters long before new separator is found. If next function was called: sScan(string,',',3,output), then output.data will point to start of test123 text and output.len will be 7 because text is 7 characters long before NULL charater is found(end of C-string).

Parameters
inputPointer to first character in C-string.
separatorCharacter used as separator between tokens.
separatorCntNumber of separators before wanted token.
outputReference to scan data output struct.
Returns
SSTD_NOK if no token was found.
SSTD_OK if token was found.
Note
C-string has to be NULL terminated.
